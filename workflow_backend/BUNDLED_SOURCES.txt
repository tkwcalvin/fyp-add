Files in this directory are copied from the main repository as the minimal
workflow dependency set, so object_insert_workflow can run self-contained.
If they diverge from upstream, run the following from your fyp root
(adjust paths for your machine):

  BE=/path/to/fyp/collage-diffusion-ui/backend
  WF=/path/to/fyp/object_insert_workflow/workflow_backend
  cp "$BE/sd_sam_pipeline.py" "$BE/paste_sticker_bbox_roi.py" "$BE/paste_sticker_roi.py" \
     "$BE/test_sd_generation.py" "$WF/"
  cp "$BE/utils/load_layer_image.py" "$WF/utils/"

For the SDXL fusion script copy notes, see ../sdxl_inpaint/BUNDLED_SOURCES.txt
